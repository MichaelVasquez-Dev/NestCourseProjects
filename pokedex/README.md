# Ejecutar el proyecto

1. Clonar el repositorio
```
git clone link_repositorio
```

2. Ejecutar en la terminal el comando
```
npm install
```

3. Tener instalado Nest CLI
```
npm install -g @nestjs/cli
``` 

4. Levantar la base de datos
```
docker-compose up -d
```


## Stack usado
* Mongo DB
* Nest JS
